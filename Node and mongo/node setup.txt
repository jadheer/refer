To begin a project:-
npm init

npm i express bcryptjs // For login auth
npm i express-handlebars // It is a type of template engine ex:- jade
npm i method-override
npm install faker // To create fake data
npm i express-fileupload // To upload file
npm i express-session // To use sessions
npm i connect-flash // To create flash messages
npm i moment // For date and time formatting
npm install passport passport-local // It is a type of authentication
npm install mongoose-url-slugs // To make create slug automatically

npm i express
npm i mongoose // For using mongoose database
npm i formidable // Used for file uploads
npm i jsonwebtoken // To generate token during login
npm i express-jwt // To secure the route / will check if the user has valid token
npm i body-parser // The data posted from form or postman cannot directly recognize by express, submitted data will show as undefined, so we installed and used this.
npm i cookie-parser // To read body of cookie like body parser for form body
npm i lodash // Used for finding updated data by finding delta
npm i morgan // To show routes called into the console, used as middleware
npm i dotenv // To read env variables
npm i express-validator // To validate the posted data // there is compatibility issue with latest version instead install // npm i express-validator@5.3.1 // for the learned way implementation
npm i uuid // To create timestamp and random string
npm i nodemon // To reload the app on every code update
npm i cors // To solve cors issue
